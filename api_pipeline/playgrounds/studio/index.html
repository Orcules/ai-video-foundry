      API Log (browser → server + PIPE + EXT + LOG)

      EXT = third-party activity: HTTP traces (EXTERNAL_API) when available, plus cost / usage lines (COST) after each billed model or API call (Gemini, Kie, ElevenLabs, …). For Studio routes that run on the server without a job (e.g. generate music, VO, character, voice design), an extra synthetic EXT line explains which providers the server will call. PIPE = internal step messages from pipeline-events. GET lines = GET /api/jobs/… (throttled when unchanged) and other REST calls from the Studio client — turn on External APIs only to hide those and focus on EXT. Server stdout: [ExternalAPI].
